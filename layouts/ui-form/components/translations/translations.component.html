Services Pipes Functions

{{'Translate statics & dynamics strings into the application. The system is based on Services, Pipes & Functions.' | translate}}

{{'How to use' | translate}}

{{'The new translation system works following these steps:' | translate}}

  1. {{'Clean your old HTML page with mltag attributes by using ' | translate}} mltag_remove.py.
  2. {{'Add new translation system by using' | translate}} translate.py.
  3. {{'Show & Generate all missing translations in database via generator.' | translate}}
  4. {{'Translate missing translations on' | translate}} {{'Translations back-office' | translate}}.
  5. {{'Generate new translations file with' | translate}} updateTranslations.sh.
  6. {{'Test all pages in your local environment.' | translate}}

{{'1. Remove old mltag in HTML' | translate}}

{{'The new translations system don\'t use anymore the mltag in HTML pages.' | translate}}
{{'Go to your local mltag-auto folder and launch this command:' | translate}} 

python mltag_remove.py ../your_folder_name/

{{'2. Add new translation system' | translate}}

{{'The new translations system use Pipes and Functions now.' | translate}}
{{'Go to your local mltag-auto folder and launch this command:' | translate}} 

python translate.py ../your_folder_name/

{{'3. Show & Generate missing translations in database' | translate}}

{{'All missing translations are detected by the translations service when we asked for it.' | translate}}
{{'You can see all theses missing translations by clicking on the Translations button on the top navigation bar.' | translate}}


{{'By clicking this button, a modal will appear and show you all missing translations' | translate }}

{{'If you close this modal, you can see all missing translations highlighted in yellow.' | translate}}

{{'By clicking the button, you can generate in database all these missing translations.' | translate}}


{{'A report will be generated after generation to inform you of the result. All saved translations are not anymore in the first part of the modal.' | translate}}
{{'The only reason a translation could be FAILED is that the translation is already in the database.' | translate}}

{{'4. Translate new translations' | translate}}

{{'Go directly to the back-office:' | translate}} {{'Translations back-office' | translate}}.
{{'You need an account to access this back-office. If you don\'t have one, ask to a developer.' | translate}}


{{'5. Generate new translations file' | translate}}

{{'When all new translations has been translated via the back-office, you can generate the new translations file for front-end application.' | translate}}

{{'Generate file with:' | translate}} updateTranslations.sh

{{'To use it, go to your multilingual folder via the terminal and launch the following command:' | translate}}

./updateTranslations.sh

{{'6. Test your pages' | translate}}

{{'Now the translations.ts file is up to date, you can see on your application the results.' | translate}}
{{'Normally, you should not have any missing translations in the modal Translations:' | translate}}

{{'Enjoy' | translate}} ;)
