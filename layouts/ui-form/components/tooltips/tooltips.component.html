{{'Tooltips' | translate}}

Directives

{{'Tooltips provides a description of a UI element.' | translate}}

{{'How to use' | translate}}

{{'Just add the directives [tooltip] and give params via an object' | translate}} {{'{'}}{{'}'}}:

  1. title: {{'\'your title here\'' | translate}} ({{'optional' | translate}})
  2. text: {{'\'your text here\'' | translate}}
  3. size: {{'\'medium\'' | translate}} ({{'optional' | translate}})
  4. autoshow: true ({{'optional' | translate}})

{{'Sizes' | translate}}

{{'There is several possible sizes you can force for the tooltips. By default, the medium size is applied.' | translate}}

  1. small: {{'limited to 50 chars. If more, medium is applied automatically.' | translate}}
  2. medium: {{'default size if not specified. max-width 320px' | translate}}
  3. large: {{'max-width 540px - for long texts' | translate}}

{{'Sizes Rules' | translate}}

{{'If you don\'t apply a size to your tooltip, some rules are auto applied:' | translate}}

  1. small: {{'if no title and max 50 chars.' | translate}}
  2. medium: {{'if title and/or more than 50 chars.' | translate}}
  3. large: {{'only if applied in size' | translate}}

{{'Autoshow rules' | translate}}

{{'A tooltip with autoshow will be visible directly on the page without over it.' | translate}}
{{'These tooltips can be closed manually by clicking the close button.' | translate}}
{{'Once a tooltip has been closed, it becomes a classic tooltip (mouseover/mouseout).' | translate}}
{{'Clicking on the element toggle the autoshow property.' | translate}}

{{'Codes' | translate}}

{{'<i class="fa fa-question-circle fa-5x" [tooltip]="{title: \'Your title here\', text: \'Your text here\', size: \'large\', size: true}"></i>'}}
