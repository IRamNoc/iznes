{{'How to use' | translate}}

{{'There is several ways to use the Form percentage component' | translate}}:

  1. {{'Form' | translate}}: {{'used to calculate the completion rate of all the fields of a form' | translate}}
  2. {{'FormGroup' | translate}}: {{'used to calculate the percentage of completion of a formgroup in a form.' | translate}}

{{'Configuration' | translate}}

{{'Form percentage use an Object to set himself.' | translate}}
{{'You need to apply this directives on a container. the progress bar will be pushed into this container.' | translate}}

  1. {{'form'}}: {{'[ form | formGroup ]'}}
  2. {{'type'}}: {{'[ square | round | border ]'}} - ({{'optional - default: border' | translate}})
  3. {{'color'}}: {{'[ info | warning | danger | success ]'}} - ({{'optional - default: success' | translate}})
  4. {{'animated'}}: {{'[true | false ]'}} - ({{'optional - default: true' | translate}})
  5. {{'stripes'}}: {{'[ true | false ]'}} - ({{'optional - default: true' | translate}})
